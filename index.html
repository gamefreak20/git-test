How git works

git --version = zie versie van git

pwd = zie waar het woord gescaned

cs [bestand] = zet een nieuwe map die moet gescanned worden

mkdir "[map]" = maak een nieuwe map aan

touch "[bestand]" = maak een nieuw bestand aan

git status = kijk welke bestanden veranderd zijn

git add [bestand] = kies het bestand dat je wilt opslaan

git commit -m [message] = commit het bestand

git checkout -- . = zet alles terug wat opgeslagen is

git clone [website] = copieer de website naar jouw folder

git remote add origin [link] = maak een link aan waar de bestanden worden opgeslagen

git push origin master = push het op de server
